CSS Selectors

    Type selectors

    matches elements by node name
    div, li, a, p

    Class selectors

    matches elements by class name
    button class=“active”

    Compound class selectors

    You can have elements in your HTML that have more than one class applied to them.
    Div element with two classes.
    div class="box yellow"

    ID selectors

    matches elements by ID name (e.g. div id=”list-1”)

    Universal selectors

    matches elements of any type (e.g. *)

    Attribute selectors

    matches elements based on the presence or value of a given attribute (e.g. a[title] matches all a elements with a title attribute)

Multiple Selectors

Same style: multiple elements

h1, h2 { font-style: italic; }

h1#heading, h2.subheading { font-style: italic; }
