CSS Selectors

  1. Type selectors -- matches elements by node name (e.g. div, li, a, p)
  2. Class selectors -- matches elements by class name (e.g. button class=“active”)

    3. Compound class selectors
    Quite often, you will have elements in your HTML that have more than one class applied to them. Consider the following example, div element with two classes. exmp: (div class="box yellow")

  3. ID selectors -- matches elements by ID name (e.g. div id=”list-1”)
  4. Universal selectors -- matches elements of any type (e.g. *)
  5. Attribute selectors -- matches elements based on the presence or value of a given attribute (e.g. a[title] matches all a elements with a title attribute)

Multiple Selectors

Same style: multiple elements

h1, h2 { font-style: italic; }

h1#heading, h2.subheading { font-style: italic; }

Examples of Each Selector

/* Type selector */
div {
background-color: #000000;
}
/* Class selector */
.active {
color: #ffffff;
}
/* ID selector */
#list-1 {
border: 1px solid gray;
}
/* Universal selector */
* {
padding: 10px;
}
/* Attribute selector */
a[title] {
font-size: 2em;
}
