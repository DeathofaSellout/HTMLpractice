CSS Selectors

    Type selectors

    matches elements by node name
    div, li, a, p

    Class selectors

    matches elements by class name
    button class=“active”

    Compound class selectors

    You can have elements in your HTML that have more than one class applied to them.
    Div element with two classes.
    div class="box yellow"

    ID selectors

    matches elements by ID name (e.g. div id=”list-1”)

    Universal selectors

    matches elements of any type (e.g. *)

    Attribute selectors

    matches elements based on the presence or value of a given attribute (e.g. a[title] matches all a elements with a title attribute)

Multiple Selectors

Same style: multiple elements

h1, h2 { font-style: italic; }

h1#heading, h2.subheading { font-style: italic; }

Combinators

Descendant selectors

Two selectors with just white space in between them.

p abbr { text-transform: uppercase; }

Direct child selector

Selects only the direct children of an element.

.menu > .is-active { background-color: #ffe0b2; }

Adjacent sibling selectors

First and Second element are styled.

h1 + h2 { font-style: italic; }

Pseudo Classes

Specifies a special state of the selected element.

a:hover {}

Pseudo Selectors

Pseudo-selectors select pseudo elements in the DOM. The two that you will use most often are the ::after and the ::before pseudo-selectors. Both of them create a pseudo-element as a child of the element to which the property applies. The ::after variation creates the child as the last child of the selected element. The ::before variation creates the child as the first child of the selected element. You can do neat things with that. For example, if you can put happy faces at the beginning of all paragraphs.

p::before {}
